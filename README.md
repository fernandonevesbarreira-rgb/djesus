# DJESUS — Sistema de Gestão de Produção

## Como colocar no Railway

1. Cria conta em railway.app
2. Clica "New Project" → "Deploy from GitHub"
3. Faz upload deste projeto
4. Railway detecta automaticamente o Node.js
5. A app fica online num link como: djesus.railway.app

## Desenvolvimento local
```
npm install
npm start
```
